# Shopers 

## About the project
Shopers Built up with ReactJs with backend ExpressJS and deployment on Vercel.

## Usage 

### Clone
```shell
git clone https://github.com/karomahlailatul/Shopers.git
```
### Installing 
```shell
npm i;
```

### Testing
```shell
npm run start;
```
